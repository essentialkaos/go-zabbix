`zabbix` is a Go package for sending metrics data to Zabbix Server 3+.

### Zabbix version support

| Zabbix Version | Support Status |
|----------------|----------------|
| `1.x`          | No             |
| `2.x`          | No             |
| `3.x`          | Yes (_Full_)   |
| `4.x`          | Yes (_Full_)   |
| `5.x`          | Yes (_Full_)   |
| `6.x`          | Yes (_Full_)   |
